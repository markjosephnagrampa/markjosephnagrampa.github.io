Restaurant Chain Management System

The Restaurant Chain Management System is a multi-chain POS website with menu management, invoicing, accounting, and table management features. The system accomodates 3 distinct user types. These are:

  • The Owner
  • Branch Manager(s)
  • Waiter(s)

This website was created as a project for my Web Systems & Technologies class in Southville International School & Colleges, and was coded using SQL Server, PHP, and the Bootstrap 3 Framework. You can access this website's codebase through this  GitHub repository.
